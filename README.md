# python-unit-converter
A simple command-line tool built in Python that converts between common units of measurement.

## Features
- Kilometers → Miles
- Celsius → Fahrenheit
- Kilograms → Pounds
- Meters → Feet
- Liters → Gallons (US)
- Loops continuously until the user chooses to exit
